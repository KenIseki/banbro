ダウンロードありがとうございます。
詳細は helpフォルダのindex.html を参照してください。

動作には Java7 の実行環境が必要です。
32ビット用実行環境(jre7 x86)でないと起動できないようです。

■これは何？
　NDS用ソフト「大合奏！バンドブラザーズDX」（以下「バンブラDX」）の曲データである「bdxファイル」を
　MIDIに変換しながら再生するツールです。
　別ツールの「BDXConverter」で変換した「BDXMLファイル（Version 1.3 以降）」や
　NDS用ソフト「大合奏！バンドブラザーズ」の曲データである「bbsファイル」も再生できます。

■何ができるの？
　・再生方法を選択して再生
　・プレイリストの作成
　・頭出し、巻き戻し、早送り、一時停止
　・演奏イメージを変更して再生
　　基本的にバンブラDXを再現し、聴きやすいように少し変更
　　バンブラDXにない演奏イメージもいくつか用意されています
　・曲の情報（アーティストなど）を記録

■音が変
　MIDIで再現困難な値は無視してるか疑似的に再現してます。
　例) sastain,CROSSエフェクト
　MIDIのチャンネル数の関係で、ドラムパートを1つ含んで最大16パートまでしか出力されません。
　バンブラ以外のツールで作成したbdxファイルやbbsファイルはうまく再生されないかもしれません。

■曲リストと情報表示で表示されている演奏時間が異なる
　初期設定の集中による遅延を考慮し先頭に1小節の空白を入れているため、情報表示の方が少し長くなります。
　また、演奏イメージによりテンポがアレンジされると変わります。
　合計時間は曲リストに表示されている演奏時間の合計です。

■曲情報取得がうまくいかない
　以下の原因が考えられます。
　・http://bbdx.maruamyu.net/api/ に接続できない
　・APIの仕様が変わった
　・原曲情報や曲情報がまだ書き込まれていない
　・受付番号が正しく設定されていない
　・BDXML Version 1.4 以降ではない（自作曲扱いになります）
　・自作曲の情報を取得しようとした

■バグを発見した
　詳しい状況を書いて連絡先へ。
