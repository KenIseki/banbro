始めに

概要

bdxをMIDIに変換しながら再生できるソフト。バンブラUploaderからダウンロードできる。
bdxの他に、BDXConverterで変換したBDXMLファイル（Version 1.3 以降）と、 bbsファイル（「大合奏！バンドブラザーズ」の曲データ）も再生できる。

動作環境

OS は Windows 7 64ビット、Java 7 でのみ動作確認をしています。（Version 2.2 から Java 7 に変わりました）
実行にはJavaの実行環境（jre7 x86）が必要です。

インストール

解凍したファイル（下記参照）を任意のフォルダに移動してください。
古いバージョンがある場合は全て上書きしてください。
※ファイル構成や名前、自動生成されるファイル（淡い色）の中身などを変更しないでください。正しく動作しなくなります。
help ヘルプで使用するファイルが入っているフォルダ
BDXPlayer.exe ツール本体
BDXPlayer.ini ツールの設定ファイル(一度起動すると自動的に作成される)
BDXPlayer.log ログファイル(エラーが発生すると自動的に作成される)(エラー解消後、削除可)
MidiSetting.ini MIDI設定ファイル(一度起動すると自動的に作成される)
MusicLibrary.xml 読み込んだファイルとプレイリストを保存しているファイル(一度起動すると自動的に作成される)

アンインストール

BDXPlayerフォルダを削除してください。 レジストリは一切使っていません。

「曲情報取得」・・・http://bbdx.maruamyu.net/api/ から曲情報を取得します。値は上書きされます。 ※接続先に迷惑がかからない程度にご使用ください。

「BDXMLバージョンを作成」・・・選択された1曲がbdxファイルならBDXMLに変換して追加します。※演奏イメージは変わりません。
任意の曲をダブルクリックするとその曲の再生が始まります。
情報表示 停止中はプレイリスト内の曲数と合計時間を表示します。

曲リストの項目

編集可の項目は、右クリックメニューの「編集」から編集できます。
項目 内容
ファイル名 ファイルがある場所。
きょくめい バンブラDX上で表示される曲名。再生時に自動的に更新。
曲名 曲の正式名称。編集可。
デフォルトは「きょくめい」と同じ。
アーティスト アーティスト。編集可。
複数の場合は「／」で区切る。
キャラ名義の場合はキャラ名の後に半角括弧で人名を囲う。「キャラ名(人名)」の様に書く。
演奏時間 テンポアレンジなしでの演奏時間。再生時に自動的に更新。
レート 曲のお気に入り度。編集可。
再生回数 曲を最後まで再生した回数。
タイアップ等 曲に関するメモ。編集可。

その他

このツール独自の演奏イメージ

以下の演奏イメージは独自に作成した演奏イメージです。バンブラDXや「Jam with the Band」にはありません。
（面白いアイディアが思いついたら連絡先へ。新たな演奏イメージとして追加されるかもしれません。）
演奏イメージ 特徴
逆再生 楽譜の時間軸を逆にします。
音色も逆再生に近い音に調整します。
楽器や音程はそのままです。
シャッフル 楽器を変更せずにシャッフルビートにします。
ブルース シャッフルビートにして、「ジャズ」と異なる楽器に変更します。
ヨナ抜き 単音パートの音階をヨナ抜き長音階(移調するとピアノの黒鍵のみになる音階)にします。
省電力 電気を使用する楽器（Ｅ・ピアノ、Ｄ・ギター、シンセリード、など）を別の楽器に変更します。
ビブラート無効、エフェクターOFF、ボリュームも少し下げ、演奏時間短縮のために演奏速度を上げます。
※この演奏イメージを選んでもPCの消費電力は変わりません。
今日のライブ バンブラDXの「今日のライブ」を再現します。
今日の日付によって演奏イメージが決定されます。
Today's Gig Jam with the Band の「Today's Gig」を再現します。
今日の日付によって演奏イメージが決定されます。
ランダム 「今日のライブ」「Today's Gig」を除いた全ての演奏イメージからランダムに一つ選びます。
